Contains initial Arduino IDE code obtained from different tutorials and used to learn how to set up and use esp32 and esp32-cam for project. 

getMACadd: get MAC address of esp32 code + tutorial from https://RandomNerdTutorials.com/get-change-esp32-esp8266-mac-address-arduino/

connectWiFiESP: connect esp32 to CMU-DEVICE. 


CameraWebServer: basic camera web server, sample code provided as part of espressif esp32-cam library. Used to set up an initial stream using HTTP.
